# 题匠 QuizForge v2

> 将 .txt 题库文件一键生成交互式 HTML 答题网页，无需服务器，浏览器直接打开即可答题。

## 功能

- **四种题型**：单选、多选、填空（支持多空）、判断，覆盖常见考试场景
- **9 个内置主题**：默认蓝、橙、绿、红、粉、黄、蓝、紫、灰，可导入/导出自定义主题
- **深色/浅色模式**：一键切换，所有界面元素自动适配
- **中英文双语**：界面语言一键切换，所有文案同步
- **实时预览**：生成前即可预览答题卡效果，主题切换实时更新
- **批量处理**：一次导入多个 .txt 文件，分别设置参数，一键全部生成
- **拖拽导入**：直接将文件拖入窗口添加题库
- **时间戳**：可在文件名前加日期，或在网页底部显示生成时间
- **自包含 HTML**：生成的 .html 文件含完整 CSS/JS，无需任何外部依赖
- **单 exe 分发**：软件本身也是单文件，双击即用，无需安装

## 快速上手

1. 按格式编写 .txt 题库文件
2. 打开软件，点击"导入题库"或拖入文件
3. 选择主题，点击"批量生成"，得到 .html 文件

## 题库格式

### 中文格式

```
# 这是一行注释，会被忽略
标题：我的数学练习

---
题型：单选
题目：35 + 28 = ?
选项：53|63|73|62
答案：1

---
题型：多选
题目：哪些是偶数？
选项：1|2|3|4
答案：2,4

---
题型：填空
题目：6 × 7 = ?
答案：42|四十二

---
题型：填空（多空）
题目：两个数，和为5，差为1
答案：2|二；3|三

---
题型：判断
题目：81 - 36 = 45 正确吗？
答案：错误
```

### 英文格式（自动识别）

```
Title: My Quiz

---
Type: Single
Question: What is 1+1?
Options: 1|2|3|4
Answer: 1

---
Type: Multiple
Question: Which are even numbers?
Options: 1|2|3|4
Answer: 2,4

---
Type: Fill
Question: What is 6×7?
Answer: 42|forty-two

---
Type: Judgment
Question: The earth is round.
Answer: True
```

### 格式规则

| 规则 | 说明 |
|------|------|
| 文件开头 | 可选 `标题：xxx` 或 `Title: xxx`，否则用文件名 |
| 分隔符 | 题目之间用 `---`（三个减号）分隔 |
| 注释 | `#` 开头的行为注释，解析时自动忽略 |
| 题型关键词 | 题型/Type，题目/Question，选项/Options，答案/Answer |
| 答案序号 | 从 1 开始计数（1 表示第一个选项） |
| 多选答案 | 逗号分隔（中英文均可），如 `1,2,3` |
| 填空单空 | 多个可接受答案用 `|` 分隔，如 `42|四十二` |
| 填空多空 | 用 `;` 或 `；` 分隔不同空，每个空内仍用 `|` 分隔 |
| 判断答案 | 支持 `正确/错误`、`true/false`、`yes/no`、`对/错` |

## 从源码构建

### 环境要求

- Node.js 18+
- Rust 1.77+

### 构建命令

```bash
build.bat
```

产物：`src-tauri/target/release/quizforge.exe`（绿色单文件，可直接分发）

### 开发模式

```bash
run_dev.bat
```
